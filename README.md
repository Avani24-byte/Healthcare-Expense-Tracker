🏥 Healthcare Expense Tracker

A simple web application to record, manage, and track healthcare expenses.  
It helps users store expense details and view them in an organized way.

🚀 Features
- Add healthcare expense records
- Store expense data in JSON format
- Backend server using Node.js
- Simple and lightweight structure

🛠 Tech Stack
- Node.js
- Express.js
- JavaScript
- JSON (for data storage)

📁 Project Structure
Healthcare-Expense-Tracker/
│
├── public/            # Frontend files
├── server.js          # Backend server
├── expenses.json      # Stored expense data
├── package.json       # Project dependencies
└── node_modules/      # Installed packages

▶ How to Run

1. Clone the repository
   git clone https://github.com/Avani24-byte/Healthcare-Expense-Tracker.git

2. Navigate to project folder
   cd Healthcare-Expense-Tracker

3. Install dependencies
   npm install

4. Start the server
   node server.js

5. Open browser and visit
   http://localhost:3000
